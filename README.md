Веб-сайт інтернет-магазину м’ясної продукції «М’ясна Лавка» (Кам’янець-Подільський).  
Проєкт надає зручний каталог товарів, фільтрацію/пошук, сторінку товару та кошик, а також інформаційні сторінки про доставку та компанію.

---

## 🧩 Опис (яку проблему вирішує)
Сайт допомагає покупцю швидко знайти потрібний товар (за видом м’яса та ціною), переглянути деталі, додати позиції в кошик і підготуватися до оформлення замовлення.  
Для бізнесу це базова онлайн-вітрина, яка може стати основою для подальшого розширення до повноцінного e-commerce (оформлення заявки/замовлення, інтеграції тощо).

---

## 📄 Структура сторінок
- `index.html` — головна (категорії, популярні товари, YouTube-слайдер, мапа)
- `shop.html` — каталог (фільтри, пошук, пагінація)
- `productPage.html` — сторінка товару (динамічні дані, “додати в кошик”, ще товари)
- `cart.html` — кошик (список, сума, рекомендації)
- `delivery.html` — доставка (умови, графік, слайдер)
- `about.html` — про нас
