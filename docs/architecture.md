# Architecture / Project Structure — «М’ясна Лавка»

Цей репозиторій містить статичний web-сайт інтернет-магазину «М’ясна Лавка».
Проєкт реалізовано на HTML + SASS/CSS + Vanilla JavaScript з використанням Bootstrap, jQuery та Slick.js.
Дані товарів зберігаються локально у файлі `productsData.js` (без бекенду/БД).

---

## 1) Pages (сторінки)

- `index.html` — головна сторінка:
  - hero-блок «Вірність традиціям якості з 2000 року»
  - категорії товарів
  - блок «Найпопулярніші товари» (рендер з JS)
  - YouTube-слайдер (Slick.js)
  - Google Maps у footer

- `shop.html` — каталог/магазин:
  - фільтр по виду м’яса
  - фільтр ціни (range + min/max)
  - тип продажу (роздріб/опт), тип виробництва
  - пошук, пагінація, скидання фільтрів (JS)

- `productPage.html` — сторінка товару:
  - назва/ціна/опис/фото з `productsData.js`
  - додаткова інформація
  - відгуки (статичний блок)
  - кнопка «Додати в кошик»
  - блок «ще товарів» (JS)

- `cart.html` — кошик:
  - рендер позицій з JS
  - підрахунок суми
  - кнопки «Продовжити покупки» / «Оформити замовлення»
  - блок рекомендацій (JS)

- `delivery.html` — доставка:
  - умови доставки (SMS-підтвердження, часові вікна)
  - слайдер фото (Slick.js)

- `about.html` — про нас:
  - статична інформація про компанію
  - спільний header/footer

---

## 2) Folders (папки)

- `js/` — JavaScript-логіка сторінок
- `sass/` — SASS-джерела стилів
- `css/` — зібрані CSS-стилі
- `font/` — шрифти
- `img.*` / зображення — медіа-ресурси (формати png/jpg/svg)

---

## 3) JavaScript modules (ключові файли)

- `js/productsData.js` — центральний набір даних товарів (масив об’єктів):
  - назва, ціна (грн/кг), вид м’яса, тип продажу (роздріб/опт), виробництво
  - фото, опис, корисні властивості
- `js/common.js` — спільна логіка:
  - навігація між сторінками
  - оновлення лічильника кошика в header/footer
  - scroll-to-top
  - burger-меню (мобільна навігація)
- `js/main.js` — головна:
  - рендер популярних товарів
  - lazy-load YouTube (по кліку)
- `js/shop.js` — каталог:
  - фільтрація, пошук, пагінація, скидання фільтрів
- `js/productPage.js` — сторінка товару:
  - підвантаження даних товару по ID/параметру
  - додавання в кошик
- `js/cart.js` — кошик:
  - рендер кошика, підрахунок суми, рекомендації
- `js/delivery.js` — доставка:
  - ініціалізація слайдера фото

---

## 4) Notes / Limitations (обмеження)

- Проєкт без бекенду: “Оформити замовлення” та “Вхід/Реєстрація” не реалізовані як повний процес.
- Дані товарів локальні (без БД), тому оновлення каталогу потребує змін у `productsData.js`.
- Є технічні моменти для покращення (як задачі/ризики): дублювання header/footer, коректні `lang`/`title`/`alt`, уникнення дубльованих CDN-підключень.
