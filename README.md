# MeatShop


Ми обрали ліцензію MIT, оскільки наш проєкт є навчальним Open Source рішенням. Ми хочемо дозволити вільне використання коду (включно з копіюванням, модифікацією та поширенням) із збереженням інформації про авторів. Також ми знімаємо з себе відповідальність за можливі проблеми та збитки, оскільки ліцензія надає код “AS IS”.
